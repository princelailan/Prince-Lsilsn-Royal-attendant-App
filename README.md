## 🚀 Setup Instructions

```bash
git clone https://github.com/princelailan/Prince-Lsilsn-Royal-attendant-App.git
cd Prince-Lsilsn-Royal-attendant-App
npm install
npm run dev (for dev)
npm run build (for production)
```

💡 For `.exe` packaging:

```bash
npm run package-win
```
